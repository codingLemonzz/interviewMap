# interviewMap

### 介绍
各类面试题八股文访问地址，欢迎补充

### 如何补充地址
直接提交pull Requests即可

### Java
##### 1. 大厂面试指北
[大厂面试指北地址](https://notfound9.github.io/interviewGuide/#/)   主要为java方向内容，包括Java基础、Redis、MySQL、Kafka以及LeetCode算法等，适合理解记忆，不适合直接背

#### 2. Java 全栈知识体系
[Java 全栈知识体系地址](https://pdai.tech/) 各方面都有比较细致讲解，但内容多且杂，适合作为类似字典类工具去查阅或日常阅读，不适合作为面试题突击背诵

### 算法
#### 1. LeetCode热门题
